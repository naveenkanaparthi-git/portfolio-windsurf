Resume placeholder

This repository does not currently include a public resume PDF.

To enable a downloadable PDF resume:
1) Export your resume as a PDF named: naveen-kanaparthi-resume.pdf
2) Place it in: public/resume/
3) Update src/data/person.json -> resumeUrl to: /resume/naveen-kanaparthi-resume.pdf

After deploying, verify:
- /resume (web page)
- /resume/naveen-kanaparthi-resume.pdf (direct download)
